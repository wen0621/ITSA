# ITSA題目

## 題目

**問題描述：**
現有一可顯示四位數字的 LED 燈一組，可根據輸入的四位數字顯示數字。

比如說，由鍵盤輸入 1234 ，則可在螢幕上顯示
![1234](https://media.discordapp.net/attachments/914196675961188443/1087287845582020638/C_ST53-1.png)

若輸入 0789 ，則顯示
![0789](https://cdn.discordapp.com/attachments/914196675961188443/1087287876691177542/C_ST53-2.png)

**輸入說明：**
輸入格式為 四個整數。

**輸出說明：**
輸出值為輸入值的矩陣數字。

**範例：**
